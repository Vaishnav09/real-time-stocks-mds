Real-Time Stock Market Data Pipeline (Modern Data Stack)
Overview

This project implements an end-to-end real-time data pipeline that ingests live stock market data, processes it using a modern data stack, and delivers analytics-ready insights for visualization.

The pipeline streams stock data through Apache Kafka, stores raw data in MinIO object storage, orchestrates ingestion with Apache Airflow, transforms data using dbt, and loads it into Snowflake for analytical querying and visualization in Power BI.

The project demonstrates how modern data engineering workflows transform raw streaming data into structured business insights.
